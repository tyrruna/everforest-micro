Everforest for the Micro text editor.

https://micro-editor.github.io/

https://github.com/sainnhe/everforest

https://addons.mozilla.org/en-US/firefox/collections/18694308/Everforest-Themes/

Copy "everforest.micro" to ~/.config/micro/colorschemes (or create a new folder named colorschemes if non-existing).

Add "export MICRO_TRUECOLOR=1" (without quotation marks) to the shell rc file (.zshrc).

Restart the Terminal.

Open the Micro text editor, press control+e and type "set colorscheme everforest" (without quotation marks) and press enter.

Done.
